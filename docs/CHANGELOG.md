# Changelog

Todas las novedades de este proyecto se documentarán aquí.

## [1.0.0] - 2025-10-30

- Paleta de colores unificada con variables CSS.
- Navbar y footer estandarizados entre páginas.
- Catálogo dividido en secciones (Hombre, Mujer, Niños, Niñas) con grid 3x.
- Estilos centralizados en `styles.css`; eliminación de CSS duplicado.
- Página de Contacto mejorada (layout responsive, inputs grandes y redes sociales: Facebook, Instagram, WhatsApp).
- Notificación tipo toast en la confirmación de compra (catálogo).
- Accesibilidad: alt en imágenes, aria-label en iconos, tipografía y tamaños aumentados.
- Corrección de enlaces, email y comentarios CSS inválidos.
