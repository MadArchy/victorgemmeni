Este documento complementa y amplía el README del proyecto. Contiene instrucciones detalladas de ejecución, estructura de archivos, contrato de funciones JavaScript, convenciones CSS, accesibilidad, pruebas y recomendaciones.

1. Resumen del proyecto

GLAMOUR NYM es una web estática para una pequeña tienda de ropa. Incluye:

- `index.html`: página principal (hero, presentación).
- `catalogo.html`: catálogo de productos dividido por secciones y modal de compra.
- `contacto.html`: información de contacto, formulario y mapa embebido.
- `styles.css`: hoja de estilos centralizada.
- `main.js`: lógica de navegación (hamburguesa) y validación del formulario de contacto.
- `catalogo.js`: lógica del modal de compra, confirmación, cálculo de total y focus trap.
